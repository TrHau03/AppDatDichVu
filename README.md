# Bước 1:

Chạy lệnh bên dưới để instal các package

```sh
# Using npm
npm i

# OR using Yarn
yarn install
```

# Bước 2:

Build và run app

### Android

```sh
# Using npm
npm run android

# OR using Yarn
yarn android
```

### iOS

```sh
# Using npm
npm run ios

# OR using Yarn
yarn ios
```
